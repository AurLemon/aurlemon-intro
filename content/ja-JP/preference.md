公の場では、自分の好みを細かく分解して語るつもりはない。好みの話になると、だいたいみんなアニメみたいな方向へ行くし、なかには小さな範囲で楽しめれば十分なものもある。ここでは思いついたことをそのまま軽く書いておく。

## アニメ

上にあるのは、これまで見たアニメと読んだ本だ。実際のところ、この数年はアニメ自体をあまり見ていないし、まして紙の本はなおさらだ。ここ一年で買った紙の本も、四六級対策みたいな教材ばかりだった。最近何を見ているのかと聞かれたら、百合、軽百合、日常系、それでほぼ終わりだと思う。昔は BG も見ていた。今も完全に見ないわけではないが、量はかなり減った。

では、なぜ最近になって急にまたアニメを見たり、GL にまで入ったのか。たぶん 2025 年 11 月ごろの話だが、理由はかなり偶然だった。私はコードを書くとき、画面右上の小窓で動画を流しておく習慣がある。背景音みたいなものだ。そうしないと、Agent の返答待ちがあまりにも遅く感じる。もちろん本気で見ているわけではなく、ほとんど聞いているだけだ。両方を行ったり来たり見ていたら、さすがに疲れすぎる。

最初は、なんとなく軽百合日常アニメの[ひびめし](https://bgm.tv/subject/531159)を選んだだけだった。思ったよりずっと好みに合っていた。では GL の沼にはどう入ったのか。そのあとで[わたなれ](https://bgm.tv/subject/524707)の MAD を見かけ、その次が[安達としまむら](https://bgm.tv/subject/282372)だった。やはりあだしまは神作だと思う。でも、どこがそんなに良いのか自分でもうまく言語化できない気がするので、ここでは書かない。変に書くと翌朝また消したくなるだろうし。ただ、自分に強く響いたのは本当だ。

あだしまの物語は、実はかなり淡々としていて、特別に大きな爆発点があるわけではない。でも、「何も起きていないように見えるのに、登場人物の心の中ではもうずいぶん遠くまで進んでいる」状態を書くのが本当にうまい。あの、近いようで遠い、拗れているのに誠実な関係性は、単純な物語の盛り上がりよりもずっと共鳴しやすい。柚原もけ先生の漫画第 8 巻は数日前、5 月 30 日に届いた。二人とも甘すぎる。タオバオで台湾版の漫画と小説 SS2 も買ったが、いつ届くのかはまだ分からない。

見てきた GL 作品の中で一番好きなものを一つ選ぶなら、やはりたぶん『あだしま』だ。見返すたびに魂が洗われる。もう一つ強く印象に残っているのは[超かぐや姫](https://bgm.tv/subject/604826)だ。作画が良く、物語の流れも気持ちよくて、気分よく見られたし泣かされた。時間があるときには今でもよく見返す。ただ、最近はそもそも「時間がある」があまりない。

ちなみに、2025 年末から 2026 年初めにかけては、暇なときにアニメを少し見る程度で、4 月以降はもうあまり見ていない。とにかく、最近買った ACGN 関連の物はあだしまと超かぐや姫ばかりだ。アニメを見るのも、ストーリーを理解しようとすると結局ちゃんと時間がかかるし、自分はその情熱を少しずつ失いつつある気がする。

というわけで最近の進捗としては、あだしまの漫画は最新話まで追いついた。樽見さん、無事に退場完了、おめでとう……のか？

## 音楽

音楽でいちばんよく聴くのは J-POP と Vocaloid だ。好きな傾向を一つ挙げるなら、たぶん math rock。ただ、音楽に関してはこれ以上気の利いた言葉が出てこない。NetEase Cloud のジャンルまとめのほうが、たぶん私よりうまく説明してくれる。芸術分野は昔から理解が難しい。J-POP の話をするなら、ヨルシカとずっと真夜中でいいのに。は外せない。私はずっと n-buna と ACAne のファンだ。DO YOU KNOW? どちらも 2021 年にハマった。

Vocaloid に関しては、固定で追っている P 主はいない。日推が流してきたものをそのまま聴く感じだ。ここで、好きな math rock + Vocaloid 曲も二つ置いておく。《[夏末递归定义](https://music.163.com/song?id=2097125583)》と《[星期零](https://music.163.com/song?id=2650358037)》。

## UI

UI デザインでは、余白とカードグリッドが好きだ。最近は特にセリフ体を使うのが好きで、ちょうど今このサイトもそうだ。たまに Emoji を視覚的なアンカーとして置いて、少しだけパンチを足すこともある 😡👊。私は「白・灰・黒を土台にして、テーマカラーを少し差し、Blur / Shadow の動きでつなぐ」ような設計がかなり好きだ。これを何というデザインスタイルと呼べばいいのかは分からない。Apple っぽさがかなり強いのかもしれない。実際、iOS のデザインはかなり好きで、特にモーション面がいい。Apple ID の顔認証決済アニメーションは本当にかっこいい。

Claude や OpenAI みたいなテック企業の公式サイトは、まさにこの方向性にかなり合っている。Clean Design はすごく Geek に見えるし、やはり大事なのは抑制だ。Claude でいちばん驚いたのは、「足し算ではなく引き算」をやっている細部だった。マウスをリンクに乗せたとき、主体を無理に光らせるのではなく、周囲の同列要素を暗くする。他の情報の重みを落として、見せたい部分を浮かび上がらせる。そのやり方は「抑制」を極限まで押し込んでいると思う。

逆に Google の Material Design は、私はあまり好きになれない。モランディ系の色を濫用していて、全体が少しごちゃごちゃして見える。初期の MD はかなり好きだったのに、今はどうしてこうなったのかと思う。

## その他

そのほかに何か好みとして取り出して話せるものがあるかと言われると、正直あまり思いつかない。プログラミングは今の自分にとって、もう趣味と言えるかどうかも怪しい。そもそもコードを書いているのも私ではなく Agent だ。Agent が作業して、私が圧をかける。もし他にまだ話題になりそうなことを挙げるなら、モンスターを飲むこととか、寝ることとかかもしれない。スマホは Xiaomi、PC は機械革命。コスパ王者と言っていい。最近は M1 Pro の MacBook を拾うべきか少し考えていたが、あまり意味がなさそうなので考えるのをやめた。

よく遊ぶゲームは [Minecraft](https://bgm.tv/subject/12596)（Java 版 ID は _Aurora_Lemon_）で、たまに Bed Wars もやる。ほかのゲームはあまりない。[Turing Complete](https://bgm.tv/subject/434205) がゲームに入るのかどうかも自分では分からない。昔は方向転換してハードウェアや組み込みをやる夢もあったが、さすがに無理だ。はは。

毎日ほぼ 4 時に寝て 11 時半に起きる。完全な夜型で、夜に作業するのに慣れている。VSCode と Obsidian のテーマはいつもライト系だ（VSCode では標準の Light Modern と GitHub Light が特に好きで、GitHub Dark も悪くない）。あまり外には出ない。誰かが本当に家の下まで来て「下に着いた」と言わない限り、たいていは理由をつけて断る。暇なときや変なアイデアが浮かんだときは、ChatGPT や Gemini と話す。Claude は封号されたので、もう聞けない。
